Анализ оттока сотрудников из компании.

Для анализа использовался датасет на 14999 записей.
Цель исследования: причины увольнения сотрудников.

Что было сделано:
- Первичный анализ данных: статистики, пропуски, распределения
- Корреляционный анализ с тепловой картой
- Сравнение сотрудников по зарплатам и департаментам
- Сравнение уволившихся и оставшихся сотрудников
- Проверка гипотезы с помощью t-теста и визуализации


Ключевые выводы:
  
  Кто увольняется: сотрудник с низкой удовлетворённостью, большим числом проектов и переработками, отсутствием повышений и низкой зарплатой;
  
  Слабые места компании: 48.8% сотрудников с низкой зарплатой, мало повышений ( чуть больше 2% за 5 лет), переработки (около 50ч/неделю в среднем), высокая текучесть кадров (23.8% уволились);
  
  Гипотезы:
  
  "Высокая зарплата -> больше работают" НЕ подтверждена (p=0.47)
  
  "Низкая удовлетворённость -> увольнение" подтверждена
  
  "Нет карьерного роста -> увольнение" подтверждена

  Общий вывод: Компания теряет людей из-за системных проблем: низкие зарплаты, нет роста, переработки.

  Использовала:
  Python, pandas, numpy, matplotlib, seaborn, scipy.stats
